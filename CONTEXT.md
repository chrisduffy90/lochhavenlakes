# Loch Haven Lakes — Project Context

## What this is
A civic reporting platform for monitoring the health of the Loch Haven lake chain in Orlando, FL.
Covers: Lake Formosa, Lake Rowena, Lake Estelle, Lake Sue, Lake Winyah.

## Current state
- Domain: lochhavenlakes.org (Namecheap)
- Repo: github.com/chrisduffy90/lochhavenlakes
- Landing page: index.html (HTML/CSS/JS + Mapbox)
- Not yet deployed to Netlify

## Stack decisions
- Frontend: Plain HTML/CSS/JS (no framework)
- Map: Mapbox GL JS (outdoors-v12 style)
- Backend (planned): FastAPI (Python)
- Database (planned): SQLite → PostgreSQL
- Hosting: Netlify (frontend), Render (backend)

## Design
- Clean, minimal, civic feel
- Fonts: DM Serif Display + DM Sans
- Colors: slate grey accent, off-white background
- Scroll animations on sections
